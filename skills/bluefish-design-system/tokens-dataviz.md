# Bluefish Dataviz Palette Scale
*Last updated: 2026-04-29*

Token path: `palette/dataviz/dv/[NN]/[stop]`

This is the **source palette** — raw color stops that feed the DV color-roles. Use it when you need a specific tint or shade for gradients, custom chart elements, or anything outside the standard role set.

**For standard chart work, use color-roles instead:** `1/color-roles/dataviz/[NN]/[property]` — see `tokens.md`.

---

## Usage notes

- `100` stop = color-roles `light` value for most series
- `500` stop = color-roles `dark` value for most series
- `main` stop = color-roles `main` value
- Use scale stops only when the standard role set (main/light/dark/contrast/onLight) isn't sufficient
